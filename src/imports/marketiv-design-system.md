# Marketiv Design System

## Brand Personality

Premium startup SaaS

Investor-ready

Trustworthy

Modern

Professional

Friendly

Clean

Not corporate ERP

Not spreadsheet software

---

## Design Philosophy

The interface should feel like a modern startup operating system.

Every screen should look consistent with Marketiv Studio System.

Prioritize clarity, spacious layouts, visual hierarchy, and premium presentation.

## Visual Rules

Use charts instead of dense tables when possible.

Avoid spreadsheet aesthetics.

Avoid ERP appearance.

Avoid Material Design appearance.

Avoid dark mode.

Maintain visual consistency with Marketiv Studio System.
